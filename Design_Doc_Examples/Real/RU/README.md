# Реальные дизайн документы на русском языке

Этот каталог содержит реальные дизайн документы ML систем на русском языке из production проектов (с удаленной конфиденциальной информацией).

## Текущие документы

1. **Vector Search Similar Queries / Выдача похожих ответов**
   - **Домен**: Веб-сервисы / ИИ приложения
   - **Проблема**: Поиск похожих пользовательских запросов для улучшения SEO и пользовательского опыта
   - **Ключевые особенности**:
     - Мультимодальный векторный поиск (текст + изображения) в едином векторном пространстве
     - Поиск похожих запросов с требованием latency < 1 секунды
     - Интеграция с MongoDB и ClickHouse
     - Использование модели jina-v4 для эмбеддингов
     - Оффлайн обработка с постепенным внедрением

## Структура документа

Каждый документ следует стандартной структуре:
1. Problem Definition
2. Metrics and Losses
3. Dataset
4. Validation Schema
5. Baseline Solution
6. Error Analysis
7. Training Pipeline
8. Features
9. Measuring and Reporting
10. Integration
