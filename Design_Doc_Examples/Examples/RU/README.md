# Примеры дизайн документов на русском языке

Этот каталог содержит примеры дизайн документов ML систем на русском языке, основанные на реальных сценариях.

## Текущие примеры

1. **Vector Search Similar Queries / Выдача похожих ответов**
   - **Домен**: Веб-сервисы / ИИ приложения
   - **Проблема**: Поиск похожих пользовательских запросов для снижения затрат на обработку ИИ и улучшения пользовательского опыта
   - **Ключевые особенности**:
     - Мультимодальный векторный поиск (текст + изображения) в едином векторном пространстве
     - Поиск похожих запросов в реальном времени с требованием latency < 1 секунды
     - Интеграция с MongoDB и векторными базами данных (ClickHouse/Elasticsearch)
     - Использование модели jina-v4 для эмбеддингов (< 0.1s latency, 6-7GB RAM)
     - Оптимизация затрат через переиспользование запросов
     - Поддержка мультимодальности: текст и изображения в едином векторном пространстве

## Структура примера

Каждый пример включает:
1. Определение проблемы (Problem Statement)
2. Пространство решений (Solution Space)
3. Требования (Requirements)
4. Дизайн системы (System Design)
5. ML компоненты (ML Components)
6. Риски и митигация (Risks and Mitigation)
7. План реализации (Implementation Plan)
8. Метрики успеха (Success Metrics)
9. Заметки и инсайты (Notes and Learnings)
